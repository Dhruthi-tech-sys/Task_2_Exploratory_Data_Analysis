# Task 2 - Exploratory Data Analysis (EDA)

## Objective
Understand the Titanic dataset using descriptive statistics and visualizations.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Steps Performed
1. Loaded Titanic dataset
2. Checked dataset shape and structure
3. Generated summary statistics
4. Analyzed missing values
5. Created histograms
6. Created boxplots
7. Visualized survival distributions
8. Performed correlation analysis
9. Generated pairplots

## Key Findings
- Females had higher survival rates.
- First-class passengers survived more frequently.
- Fare contains several outliers.
- Age and Fare distributions are skewed.

## Result
Successfully performed Exploratory Data Analysis on the Titanic dataset and identified important patterns and relationships.
